# RallyeLecture-site-web

Site web du rallye lecture pour les élèves.

Le projet regroupe 2 grandes parties :

* Connection des élèves au site.
* Ajout de livre à la base de données.

Les outils mis en oeuvre : 

* CodeIgniter
* Php
* Html5
* Git
